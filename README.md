# A-cyclic-and-predictable...supplementary
Supplementary material for the manuscript entitled "A cyclic and predictable stressor enhances exploitation and reduces exploration: The moon's phase effect on wild gerbils"
The repository contains the following files:

#1 revised_analysis.r - the R code analyzing the results of the research.
#2 final_analysis.Rproj - the R Studio project file.
#3 summary_maze.csv - dataset containing the maze experiment data.
#4 summary_control.csv - dataset containing the control experiment data.
#5 combined_df.csv - dataset containing both the maze & control experiments data.
#6 solving_time.csv - dataset containing the time it took the gerbils to solve each maze level.
