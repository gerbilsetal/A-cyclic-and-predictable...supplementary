# A-cyclic-and-predictable...supplementary
Supplementary material for the manuscript entitled "A cyclic and predictable cue intricately affects the exploration-exploitation tradeoff – The moon's phase effect on wild gerbils"
The repository contains the following files:

#1 DataAnalysis.r - the R code analyzing the results of the research.
#2 summary_maze.csv - dataset containing the maze experiment data.
#3 summary_control.csv - dataset containing the control experiment data.
#4 combined_df.csv - dataset containing both the maze & control experiments data.
